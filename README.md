# こいこい（花札）

HTML5 (canvas) と JavaScript でこいこいを作ってみる。

## こいこいとは？

  こいこいとは、2人で遊ぶ花札の競技の一つである。手札の花と場札の花を合わせてそ
  れを自分の札とし、獲得した札で出来役を成立させて得られる得点を競う、「めくり」
  系のゲームであるが、札の点数を計算しないこと、「こい」による競技継続の選択があ
  ることなどに特徴がある。

  -- Wikipedia
